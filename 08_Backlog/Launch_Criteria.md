# FreediveGo Launch Criteria

## Purpose

Define the exact conditions required before launching Closed Beta.

---

## MVP Launch Decision

Closed Beta may start only when all Critical items are completed.

---

# CRITICAL

## Authentication

Status: Required

Checklist:

- [ ] Registration works
- [ ] Email confirmation works
- [ ] Login works
- [ ] Logout works
- [ ] Password reset works

---

## Profiles

Status: Required

Checklist:

- [ ] Personal profile opens
- [ ] Public profile opens
- [ ] Profile editing works
- [ ] Profile data saves correctly

---

## Messaging

Status: Required

Checklist:

- [ ] One unified messaging system
- [ ] Messages send correctly
- [ ] Messages receive correctly
- [ ] Conversation list works
- [ ] No duplicate chat windows

---

## Spots

Status: Required

Checklist:

- [ ] Spot card opens
- [ ] Spot reviews display
- [ ] Spot submission works
- [ ] Spot moderation works

---

## Administration

Status: Required

Checklist:

- [ ] Moderation Inbox works
- [ ] Spot Requests work
- [ ] Published Spots work
- [ ] Reviews section works
- [ ] Users section works

---

# HIGH PRIORITY

## Public Profile

Checklist:

- [ ] Records section
- [ ] About section
- [ ] My Instructor block
- [ ] Achievements section

---

## Spot Card

Checklist:

- [ ] Reviews block
- [ ] Schools block
- [ ] Instructors block
- [ ] Tours block

---

# NOT REQUIRED FOR CLOSED BETA

- Payments
- Mobile apps
- Marketplace
- Equipment rental
- Instructor CRM
- Advanced rankings
- AI coaching

---

# Success Criteria

Closed Beta is successful if:

- 20-50 users can use the platform
- Registration works without assistance
- Messaging works reliably
- Spot workflow works
- Moderation workflow works
- No critical bugs remain

---

# Final Approval

Required approvals:

- Executive Director
- Product Manager
- Solution Architect
- QA Engineer

---

# Main Rule

Stability before growth.
