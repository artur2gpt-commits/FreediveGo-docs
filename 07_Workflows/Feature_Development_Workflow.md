# FreediveGo Feature Development Workflow

Before any implementation:

1. Read Vision.md
2. Read FreediveGo_Overview.md
3. Read Roadmap_2026.md
4. Read Approved_Decisions.md
5. Read FreediveGo_Core_Decisions.md
6. Read June_2026.md

Review by:
- Product Manager
- Solution Architect
- UX Reviewer
- QA Engineer

Implementation:

1. Create plan
2. Implement
3. Self-review
4. QA review
5. UX review
6. Security review if required

Deliver:

- Changed files
- What was done
- What to test
- Risks
- Next step
