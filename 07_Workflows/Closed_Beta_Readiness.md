# FreediveGo Closed Beta Readiness

## Goal

Prepare FreediveGo for closed beta with 20-50 real freedivers.

---

## Required Before Closed Beta

### Authentication

- Registration works
- Email confirmation works
- Login works
- Logout works

### Profiles

- Personal profile opens
- Public profile opens
- Profile data displays correctly

### Messaging

- One unified chat system
- Messages can be sent
- Conversations list works
- No duplicate old chat windows

### Spots

- Spot card opens
- Reviews display
- Add spot flow works
- Spot moderation works

### Admin Panel

- Moderation Inbox works
- Spot Requests work
- Published Spots work
- Reviews section works
- Users section works

---

## Beta User Group

Target:

- 20-50 freedivers
- instructors
- active community members
- trusted testers

---

## Success Criteria

Closed beta is successful if:

- Users can register without help
- Users can open and use profiles
- Users can send messages
- Users can add or review spots
- Admin can moderate content
- No critical bugs block normal usage

---

## Not Required For Closed Beta

- Payments
- Mobile apps
- Advanced ratings
- Full marketplace
- Large-scale marketing

---

## Main Rule

Closed beta requires stability, not feature completeness.
